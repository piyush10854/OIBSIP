# Random Password Generator (Python Project)

## Objective

The objective of this project is to develop a simple command-line application in Python that generates secure and random passwords based on user-defined criteria such as length and character types. This project helps beginners understand randomness, string handling, and user input in Python.

---

## Steps Performed

1. **User Input**

   * Prompted the user to enter the desired password length.
   * (Optional) Asked whether to include letters, numbers, and symbols.

2. **Character Set Creation**

   * Used Python's `string` module to define:

     * Alphabets (A–Z, a–z)
     * Digits (0–9)
     * Special symbols (!, @, #, etc.)

3. **Random Password Generation**

   * Combined selected character sets.
   * Used the `random` module to generate a password by selecting random characters.

4. **Display Output**

   * Printed the generated password to the user.

---

## Tools & Technologies Used

* **Programming Language:** Python
* **Libraries Used:**

  * `random` → for generating random characters
  * `string` → for predefined character sets

---

## Outcome

* Successfully created a functional password generator.
* The program generates strong and random passwords instantly.
* Users can customise password length and character types.
* Improved understanding of:

  * Python basics
  * Randomization
  * String manipulation
  * Conditional logic

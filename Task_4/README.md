# Basic Weather App (Python)

## Objective

The objective of this project is to build a simple command-line weather application using Python that fetches real-time weather data for a user-specified location. This project helps beginners understand API integration, JSON data handling, and user interaction.

---

## Steps Performed

1. Collected user input (city name).
2. Connected to a weather API (OpenWeatherMap).
3. Sent a request to fetch weather data.
4. Parsed the JSON response received from the API.
5. Extracted key information such as:

   * Temperature
   * Humidity
   * Weather condition
6. Displayed the results in a readable format.
7. Added basic error handling for invalid city names.

---

## Tools & Technologies Used

* **Python** (Programming Language)
* **Requests Library** (for API calls)
* **OpenWeatherMap API** (for weather data)
* **Command Line Interface (CLI)**

---

## Features

* User-friendly input system
* Real-time weather data
* Displays:

  * Temperature (°C)
  * Humidity (%)
  * Weather description
* Basic error handling

---

## Outcome

The project successfully demonstrates how to:

* Fetch real-time data from an external API
* Work with JSON data in Python
* Build a simple and interactive command-line application

This project strengthens foundational programming skills and provides a base for building more advanced applications like GUI-based weather apps or mobile apps.

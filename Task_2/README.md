# BMI Calculator 

## Objective

The objective of this project is to build a simple Body Mass Index (BMI) calculator using Python. This application helps users determine their BMI based on their weight and height, and classifies it into standard health categories.

---

## Tools & Technologies Used

* **Python** (Core Programming Language)
* **Command Line Interface (CLI)**
* Basic Python concepts:

  * Variables
  * Input/Output
  * Conditional Statements (if-else)

---

## Steps Performed

1. Took user input for:

   * Weight (in kilograms)
   * Height (in meters)

2. Applied the BMI formula:

   [
   BMI = \frac{Weight\ (kg)}{Height^2\ (m^2)}
   ]

3. Calculated BMI using Python arithmetic operations.

4. Used conditional statements to classify BMI into categories:

   * Underweight
   * Normal weight
   * Overweight
   * Obese

5. Displayed the BMI value along with the corresponding category.

---

## Outcome

* Successfully developed a working BMI calculator.
* The program accurately calculates BMI based on user input.
* It provides a clear health classification based on standard BMI ranges.
* Demonstrates understanding of basic Python programming concepts.

---

## How to Run the Project

1. Install Python on your system.

2. Save the Python file as `bmi_calculator.py`.

3. Open a terminal or command prompt.

4. Run the program using:

   ```
   python bmi_calculator.py
   ```

5. Enter your weight and height when prompted.
